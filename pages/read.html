Leer registros en la base de datos

Pasos para leer registros existentes en una base de datos:

  1. Haber establecido conexion con el servidor MySQL.
    Si no ha realizado la conexión puede revisar el siguiente material - Conexión a una DB.
  2. Para el siguiente ejercicio tenemos 3 registros previamente insertados en la tabla usuarios
  3. Para listar y leer los registros de una tabla es necesario ejecutar la consulta SELECT * FROM tabla_nombre;
  4. Ahora solo tenemos que codificar la query haciendo uso de PDO
  5. Una vez tengamos codificada la sentencia o consulta, tenemos que recorrer los resultados con un foreach

Con esto ya estariamos leyendo y visualizando los registros en la tabla usuarios de la base de datos, dichos datos pueden organizarsen de la manera en que queramos, en este caso fue organizado en un listado, pero podriamos ponerlo en una tabla o en un parrafo.
